## Quick Reference: SQL Queries Used

### Get Overall Stats
```sql
SELECT 
  COUNT(*) as total_feedback,
  AVG(rating)::numeric(3,2) as avg_rating,
  COUNT(DISTINCT user_id) as unique_testers
FROM user_feedback
WHERE created_at >= '[start_date]' AND created_at <= '[end_date]';
```

### Get Category Breakdown
```sql
SELECT 
  category,
  COUNT(*) as submissions,
  AVG(rating)::numeric(3,2) as avg_rating
FROM user_feedback
WHERE created_at >= '[start_date]'
GROUP BY category
ORDER BY submissions DESC;
```

### Get Rating Distribution
```sql
SELECT 
  rating,
  COUNT(*) as count,
  ROUND(COUNT(*)::numeric / (SELECT COUNT(*) FROM user_feedback) * 100, 1) as percentage
FROM user_feedback
GROUP BY rating
ORDER BY rating DESC;
```

### Get Beta Tester Activity
```sql
SELECT 
  user_id,
  name,
  topics_studied,
  quizzes_taken,
  feedback_count,
  testing_status
FROM beta_testers
ORDER BY feedback_count DESC;
```

### Get Recent Comments
```sql
SELECT 
  user_id,
  category,
  rating,
  comments,
  created_at
FROM user_feedback
WHERE comments IS NOT NULL
ORDER BY created_at DESC
LIMIT 20;
```
